# 欢迎来到我的博客！

这是我用 Next.js 创建的第一篇博客文章。很高兴能在这里分享我的想法和经验。

## 为什么选择 Next.js？

Next.js 为构建博客提供了诸多优势：

- **静态导出**：非常适合部署到 GitHub Pages
- **App Router**：基于 React Server Components 的现代路由方案
- **TypeScript**：开箱即用的类型安全支持
- **Tailwind CSS**：快速构建用户界面

## 开始使用

用 Next.js 构建博客非常简单，你可以按照以下步骤创建自己的博客：

1. 初始化一个新的 Next.js 项目
2. 在 `next.config.js` 中配置静态导出
3. 将博客文章创建为 Markdown 文件
4. 部署到 GitHub Pages
